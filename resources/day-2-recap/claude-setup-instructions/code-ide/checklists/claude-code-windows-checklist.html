Claude Code Setup — Windows

Open PowerShell (Start → "PowerShell"). Run commands one at a time.

Before you start
Confirm winget works: winget --version
If it errors: search "App Installer" in the Microsoft Store, install it, reopen PowerShell
Install
Install Git — winget install Git.Git
Install Node — winget install OpenJS.NodeJS
Install VS Code — winget install Microsoft.VisualStudioCode
Install Python — winget install Python.Python.3.13
Close PowerShell, reopen, then verify: git --version  ·  node --version  ·  npm --version  ·  python --version  ·  code --version
If npm fails: Set-ExecutionPolicy RemoteSigned -Scope CurrentUser, type Y, reopen PowerShell
If python opens the Store: Settings → Apps → App execution aliases → turn off python.exe
Install Claude Code
irm https://claude.ai/install.ps1 | iex
Close PowerShell, reopen, then claude --version
claude doctor — no errors
Clone the repo
cd $env:USERPROFILE; mkdir Development; cd Development
git clone https://github.com/bok-learning-lab/claude-code-20260519.git
Open in VS Code
cd claude-code-20260519; code .
"Do you trust the authors?" → Yes
Log in to Claude
VS Code: Terminal → New Terminal (Ctrl+`)
claude — follow login prompts, sign in via browser
Test: "What files are in this project?"
VS Code extension
Ctrl+Shift+X → search Claude Code → install (publisher: Anthropic)
Click the ✱ icon in the Activity Bar → sign in → test a message
