Claude Code Setup — macOS

Open Terminal (Cmd+Space → "Terminal"). Run commands one at a time.

Before you start
Run brew --version — if it errors, install Homebrew from the full guide, then reopen Terminal
Install
Install Git — brew install git
Install Node — brew install node
Install VS Code — brew install --cask visual-studio-code
Install Python — brew install python@3.13
Close Terminal (Cmd+Q), reopen, then verify: git --version  ·  node --version  ·  npm --version  ·  python3 --version  ·  code --version
If code fails: VS Code → Cmd+Shift+P → "Shell Command: Install 'code' command in PATH," reopen Terminal
Install Claude Code
curl -fsSL https://claude.ai/install.sh | bash
Close Terminal (Cmd+Q), reopen, then claude --version
claude doctor — no errors
Clone the repo
cd ~ && mkdir Development && cd Development
git clone https://github.com/bok-learning-lab/claude-code-20260519.git
Open in VS Code
cd claude-code-20260519 && code .
"Do you trust the authors?" → Yes
Log in to Claude
VS Code: Terminal → New Terminal (Ctrl+`)
claude — follow login prompts, sign in via browser
Test: "What files are in this project?"
VS Code extension
Cmd+Shift+X → search Claude Code → install (publisher: Anthropic)
Click the ✱ icon in the Activity Bar → sign in → test a message
