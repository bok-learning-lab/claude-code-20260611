Claude Code — commands worth knowing A short reference to the commands most useful in everyday work.

Type / in a session to see what's available. A command is only recognized at the start of a message; text after it is passed as arguments.

Notation: <arg> = required  ·  [arg] = optional.

Command What it does
/init Generate a starter CLAUDE.md for a repo so Claude understands the project. Run this first in any new codebase.
/compact [instructions] Summarize the conversation to free up context without losing the thread. Optionally focus the summary.
/context [all] Visualize where your context window is going, with warnings and optimization tips. Pair with /compact.
/rewind Roll code and/or conversation back to an earlier checkpoint, or summarize from a point. aliases: /checkpoint · /undo
/resume [session] Return to an earlier conversation by ID/name, or open the picker. alias: /continue
/branch [name] Fork the current conversation, preserving the original to return to later. alias: /fork
/btw <question> Ask a quick side question without bloating the main conversation history.
/usage Session cost, plan limits, and activity stats. aliases: /cost · /stats
