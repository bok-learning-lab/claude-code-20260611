texts-and-translation / folder index

A scaffolded demonstration of LLM-assisted close work for disciplines whose primary sources are not in English. Start with summary.md; everything else is here for browsing.

Top-level

  • summary.md — what this project is, how we built it, what you can translate it to
  • CLAUDE.md — project-level instructions loaded by Claude Code on session start
  • index.md / index.html — this file

inputs/

Two demo corpora, each set up as a standalone target.

  • inputs/the-odyssey/
    • odyssey_homeric_greek.xml — Homeric Greek in TEI/XML (PerseusDL, Murray edition)
    • inputs/the-odyssey/translations/ — 14 translations across 6 languages
      • English: Bryant, Butcher/Lang, Butler, Cotterill, Cowper, Merry/Riddell, Monro, Pope, Alford
      • Greek: Polylas (Modern Greek)
      • Latin: odyssea-latin
      • French: Calbet
      • Spanish: Segala Estalella
      • Swedish: Sjöström
  • inputs/early-modern-sanskrit/
    • jagannatha-rasagangadhara.htm — Ānana 1 of the Rasagaṅgādhara (17th c.) in Devanāgarī from GRETIL

operations/

Prompts and skills.

  • operations/deep-research-prompt.md — prompt that commissioned the background research at outputs/research-result.md
  • .claude/skills/
    • split-into-books/ — utility: pre-splits translation files into per-book files
    • show-passage/ — display a passage with Greek original + chosen translations
    • identify-figures-greek/ — identify rhetorical figures in a Homeric Greek passage
    • identify-figures-sanskrit/ — identify candidate alaṃkāras in a Rasagaṅgādhara passage

outputs/

Produced artifacts. Skill outputs preserve the corpus nesting used in inputs/.

  • outputs/the-odyssey/
    • figures-greek/ — figure-identification outputs against the Homeric Greek corpus
    • passages/ — /show-passage outputs
  • outputs/research-result.md — background research on the project's place in the philological / translation-theory traditions

To run end-to-end: open this folder in Claude Code, run /split-into-books all once to pre-split the translation files, then /show-passage 1.1-10 on the proem to see Wilson next to Lattimore next to Pope; then /identify-figures-sanskrit on an illustrative verse from the Rasagaṅgādhara whose alaṃkāra Jagannātha himself names.
