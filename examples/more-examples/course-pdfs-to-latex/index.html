Course PDFs to a LaTeX template

A recipe: turn a course's Word-exported worksheet and homework PDFs into clean .tex files that follow a house template, with accessibility baked into the template. Read summary.md for the full story, or CLAUDE.md for session instructions.

The move worth noticing: reconstruct a missing dependency from how it is used so you stay unblocked — then keep the provided package untouched and own your additions (the visual layer and accessibility) in a companion file, so each concern has one source of truth.

inputs/ (read-only source)

  • handoff.md — the original anonymized request.
  • worksheet-template.tex — worksheet style to match.
  • problem-set-template.tex — problem-set style to match.
  • Worksheets/03-First-Order-Linear-Differential-Equations.pdf — representative worksheet source.
  • Homework/03-First-Order-Linear-Differential-Equations.pdf — representative homework source.

operations/ (the process)

  • handout.sty — the provided environment package (used as-is).
  • coursestyle.sty — the companion visual + accessibility layer.
  • convert-worksheet-prompt.md
  • convert-homework-prompt.md
  • accessibility-audit-prompt.md
  • build.sh — rebuild every example.

outputs/ (rebuildable)

  • Worksheets/ — converted worksheets (.tex + .pdf).
  • Homework/ — converted problem sets (.tex + .pdf).
  • ACCESSIBILITY.md — the accessibility audit and gap.

To run end-to-end

Install tectonic (and pdftotext/poppler). Extract a source PDF with pdftotext -layout, follow the matching prompt in operations/prompts/ to write a .tex into outputs/, then compile with tectonic. To rebuild all committed examples, run operations/scripts/build.sh.
