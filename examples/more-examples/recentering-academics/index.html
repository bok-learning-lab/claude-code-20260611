recentering-academics / folder index

A worked example of using Claude Code to turn Bok Center guidance, Harvard institutional context, and external grading research into source-grounded curricular recommendations for a department or an individual course. Start with summary.md; everything else is here for browsing.

Top-level

  • summary.md — what this project is, how we built it, what you can translate it to
  • CLAUDE.md — project-level instructions loaded by Claude Code on session start
  • index.md / index.html — this file

inputs/

Three source layers (Bok normative, Harvard urgency, external research) plus two targets (concentration profiles and a sample syllabus).

  • inputs/bok-advice/ — Bok Center's Recentering Academics framework
    • recentering-academics.md / .html — overview
    • course-policies.md / .html
    • designing-assignments.md / .html
    • facilitating-class-sessions.md / .html
  • inputs/harvard-context/ — Harvard's institutional case for change
    • harvard-grade-report.md / .pdf — Dean Claybaugh's October 2025 grade report (the data)
    • proposal-grading-policies.md / .pdf — February 2026 Subcommittee proposal (20%+4 cap, ave-percentile-rank)
    • Supporting coverage: Gazette plan-explained, Gazette leading-fas, Crimson on the rigor problem, Crimson 15Q with Claybaugh, 1636 on grade-inflation, 1636 on PhD cuts, Inside Higher Ed on the cap
  • inputs/research/ — external empirical evidence on grading
    • overview.md — synthesis
    • Sadler_2005 / .pdf — criterion- vs norm-referenced grading
    • Jonsson & Svingby_2007 / .pdf — analytic rubrics, exemplars, rater training
    • Butcher et al_2014 / .pdf — the Wellesley mean cap
    • Bar et al_2009 / .pdf — Cornell median-posting experiment
  • inputs/fields-of-concentration/ — targets for the concentration operation
    • fields-of-concentration.pdf — official source
    • primary/ — one markdown file per primary field
    • secondary/ — one markdown file per secondary field
  • inputs/syllabus/ — target for the syllabus operation
    • sample-syllabus.docx — LING 5312 (Language & Politics)

operations/

Two prompts, one per operation.

  • 01-concentration-recommendations-prompt.md — takes one Fields-of-Concentration profile; produces concentration-level recommendations
  • 02-syllabus-redesign-prompt.md — takes one course syllabus; produces a syllabus-level redesign

outputs/

Worked examples for the Linguistics demo.

  • linguistics-concentration-recommendations.md — recommendations for the Linguistics concentration
  • sample-syllabus-redesign.md — redesign of LING 5312 (Language & Politics)

To run the demo: open this folder in Claude Code, then invoke either prompt against the Linguistics target (or substitute a different concentration profile or syllabus). Existing outputs are overwritten in place rather than producing parallel copies.
