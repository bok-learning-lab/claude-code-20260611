# Memo template — negative-case audit

Use this structure verbatim. Replace bracketed placeholders.

---

# Negative-case audit — [short claim, headline form]

**Claim audited:** [verbatim claim as the user stated it]

**Corpus:** [path to corpus directory, e.g. `inputs/transcripts/`], N = [number of transcripts read].

**Date of audit:** [YYYY-MM-DD]

## Scope

State the claim's scope condition (which respondents the claim is meant to apply to). List each respondent and whether they are in scope. Out-of-scope respondents are not negative cases — they are simply not the population the claim is about.

- **R001 Tasha** — [in scope / out of scope, with one-sentence reason]
- **R002 Marisol** — [in scope / out of scope, with one-sentence reason]
- **R003 Carla** — [in scope / out of scope, with one-sentence reason]
- **R004 Denise** — [in scope / out of scope, with one-sentence reason]

## Negative cases

For each in-scope respondent whose evidence cuts against the claim, write one subsection.

### [R00X Pseudonym]

**Verbatim excerpts:**

> [exact quote from the transcript, with stage directions like `[laughs]` preserved] — [Rxxx Lyy](inputs/transcripts/Rxxx-Transcript.md#Lyy)

> [another verbatim excerpt if relevant] — [Rxxx Lyy-zz](inputs/transcripts/Rxxx-Transcript.md#Lyy-Lzz)

**Reasoning:** 2-4 sentences explaining *how* this evidence cuts against the claim. Name the distinction the claim was missing if there is one (e.g., the claim conflates two dimensions; the claim assumed a mechanism this case does not run on).

[repeat the subsection for each additional negative case]

## In-scope cases that support the claim

Briefly list (one sentence each) the in-scope respondents whose evidence supports the claim, so the memo represents the whole in-scope corpus, not just the disconfirming subset.

- **R00X Pseudonym** — [one-sentence summary of supporting evidence, with a representative line reference].

## Implications for the theory

Recommend one of:

- **Abandon** — the claim is wrong.
- **Narrow in scope** — the claim holds for a defined subset; state the boundary explicitly.
- **Keep with explicit treatment** — the claim mostly holds; the disconfirming case(s) must be named and engaged in the eventual write-up, not buried.

Give one paragraph of reasoning for the recommendation. The recommendation is for the analyst to evaluate, not a verdict.
