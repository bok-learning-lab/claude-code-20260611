interview-coding / folder index

A worked example of LLM-assisted qualitative interview coding (Deterding & Waters 2018). Start with summary.md for context; everything else is here for browsing.

Top-level

  • summary.md — what this project is, how we built it, what you can translate it to
  • CLAUDE.md — project-level instructions loaded by Claude Code on session start
  • index.md / index.html — this file

inputs/

The synthetic four-respondent corpus and the interview protocol.

  • inputs/README.md — design rationale for the synthetic corpus
  • inputs/protocol.md — semistructured interview guide
  • inputs/attributes.csv — demographic/contextual attributes spreadsheet
  • inputs/transcripts/
    • R001-Transcript.md — Tasha (textbook instrumental)
    • R002-Transcript.md — Marisol (textbook expressive)
    • R003-Transcript.md — Carla (mixed, self-aware)
    • R004-Transcript.md — Denise (construct-validity stress test)

operations/

Prompts and skills that run against inputs/.

  • operations/deep-research-prompt.md — prompt used to commission the background research in outputs/research-result.md
  • operations/skills/
    • index-transcript/ — Waters' Stage 1 indexing
      • SKILL.md
      • reference/protocol-mapping.md
      • reference/diff-check.md
      • examples/ — four indexed transcripts + four coverage reports
    • find-negative-cases/ — Waters' Stage 3 validation
      • SKILL.md
      • reasoning-guide.md
      • memo-template.md
      • examples/ — worked memos for the two demo claims
    • methods-paragraph/ — transparency artifact generator
      • SKILL.md
      • examples/ — empty-project and full-project sample outputs

outputs/

Produced artifacts. Generated by running the skills against the corpus, or by deep research.

  • outputs/research-result.md — background research on Deterding & Waters' position in the qualitative-methods literature (markdown)
  • outputs/research-result.html — same, as a styled HTML page

To run the demo end-to-end: open this folder in Claude Code, then /index-transcript on each transcript, /find-negative-cases on the demo claim about instrumental mothers, /methods-paragraph against the project state.
