# inputs/papers

Source readings for the course — assigned papers, book chapters, problem sets, syllabi (`.pdf`, `.md`, `.docx`). Claude reads these to **ground** the materials it generates (handouts, outlines, study guides) in what the course actually assigns, rather than in its general knowledge.

Convert anything proprietary (`.docx`, `.pdf`) to markdown where you can — the pipeline works best on simple text.
