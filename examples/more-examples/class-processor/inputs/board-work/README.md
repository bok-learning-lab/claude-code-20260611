# inputs/board-work

Photos and scans of live board work — whiteboard or blackboard, handwritten derivations, diagrams, marginalia — captured during a session. Drop image files here (`.jpg`, `.png`, `.heic`) or PDF scans.

These are the inputs Claude reads with **vision**: it transcribes the equations and labels, describes the layout and teaching sequence, and turns a photo into structured text the rest of the pipeline can use. See [operations/board-vision-prompt.md](../../operations/board-vision-prompt.md).
