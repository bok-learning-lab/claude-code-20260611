# inputs/audio

Raw session audio — lecture or workshop recordings (`.m4a`, `.mp3`, `.wav`). The first step is transcription into a diarized markdown transcript that lands in [inputs/transcripts/](../transcripts/), which the key-takeaways prompt then distills.

(Claude Code doesn't run speech-to-text itself — transcribe with your tool of choice, then drop the resulting markdown in `inputs/transcripts/`. The audio stays here as the source of record.)
