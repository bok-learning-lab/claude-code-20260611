# CLAUDE.md

Project-level instructions loaded when Claude Code starts in this folder.

## What this project is

A worked example in progress: drafting routine administrative email (announcements, scheduling notes, follow-ups) from source material in `inputs/`. The prompts and skills that do the work live in `operations/` and `.claude/skills/`.

## Conventions

- `inputs/` is read-only source material. Don't modify it.
- Generated drafts go in `outputs/`. Regenerating overwrites.
- No emojis in any file.
