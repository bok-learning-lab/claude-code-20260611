Project-level instructions loaded when Claude Code starts in this folder.

## What this project is

A worked example in progress: turning rough workshop material from `inputs/` into print-ready, house-style handouts in `outputs/`. The prompts and skills that do the work live in `operations/` and `.claude/skills/`.

## Conventions

- `inputs/` is read-only source material. Don't modify it.
- Generated artifacts go in `outputs/`. Regenerating overwrites.
- No emojis in any file.
